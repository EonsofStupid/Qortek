# Public Boundary

Qortek is public. qortex release work remains private until intentionally published.

## Allowed in this repo

- stable public contracts
- RRO models
- store/vector/fork/mesh/preflight traits
- docs
- examples
- lightweight placeholders for adapter crates
- public adapter implementations that do not require private code

## Not allowed by default

- private qortex source
- private waiver text or legal records
- secrets
- local machine paths
- vendor code copied without explicit license review
- production claims without evidence

## Release principle

Public Qortek should be useful even if private qortex is unavailable.
