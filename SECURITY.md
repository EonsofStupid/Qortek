# Security

Qortek is pre-release. Do not use it as a security boundary yet.

## Supported versions

No stable supported versions exist until the first tagged release.

## Reporting

Open a private disclosure channel before publishing details for issues involving:

- auth bypass;
- remote code execution;
- secret leakage;
- unsafe mesh transport behavior;
- persistence corruption;
- cross-workspace memory bleed.

## Security model goals

Qortek is intended to support capability-scoped AI-to-AI access, workspace-level policy, durable audit trails, and explicit adapter boundaries. Those goals are not yet implemented end-to-end.
